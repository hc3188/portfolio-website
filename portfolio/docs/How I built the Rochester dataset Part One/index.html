We'll be using Selenium so we'll import that along with our requests library. Now we'll instruct selenium to go to the first page of search results for therapists in Rochester.

import requests
from selenium import webdriver
driver = webdriver.Chrome()
driver.get('https://www.psychologytoday.com/us/therapists?search=rochest')
import time

We're going to use a while loop to tell Selenium to run through all the pages of search results and pull the link to each therapist's profile. The urls will be added to list_of_links.

In the loop below, i < 25 corresponds to the number of results pages. So, we're telling our scraper to stop after the last page of results. When I did this back in early 2021 there were 24 results pages, that may have changed since.

As it loops through the results, it's going to identify profile boxes by their xpaths and pull out the link. We need to try/except so that it skips over the ad that pops up in the middle of the page.

We'll set sleep at 10 so we don't get booted. Finally, we're telling Selenium to click the next button when it gets to the bottom of the page and increase i by one every time it does so.

list_of_links = []

i = 1 
while i < 25:
    results = driver.find_element_by_xpath('/html/body/div[3]/div[3]/div[1]/div')
    profiles = results.find_elements_by_xpath('/html/body/div[3]/div[3]/div[1]/div/div')
    
    for profile in profiles:
        try:    
            names = profile.find_element_by_class_name('row')
            a_tag = names.find_element_by_tag_name('a')
            links = a_tag.get_attribute('href')
            list_of_links.append(links)
            print(links)
        except:
            print('')
    time.sleep(10)
    driver.find_elements_by_class_name('btn-next')[1].send_keys('\n')
    i += 1
